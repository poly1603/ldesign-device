# Examples

这个目录包含了 `@ldesign/device` 的使用示例。

## 示例项目

- **vanilla-js** - 原生 JavaScript 使用示例
- **vue-example** - Vue 3 使用示例

## 运行示例

每个示例项目都有自己的 README 文件，包含详细的运行说明。

### 快速开始

1. 进入示例目录
2. 安装依赖
3. 启动开发服务器

```bash
# 进入示例目录
cd examples/vanilla-js
# 或
cd examples/vue-example

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
